# Data Mining Course
This repository contains laboratory notebooks for my Data Mining course (Data Science AGH).

The course covers key concepts and methods for extracting knowledge from data, including:
- Dimensionality reduction (e.g., PCA, SVD)
- Data preprocessing (cleaning, transformation, feature extraction)
- Classification (decision trees, k-NN, rule-based methods, model evaluation)
- Clustering techniques (partitioning, hierarchical, density-based)
- Association rule mining (Apriori, FP-Growth and variants)
- Time series analysis (segmentation, pattern detection, prediction)
- Data warehousing (OLTP vs OLAP, schema design)
- Social network analysis
- Text processing methods
- Big Data and parallel processing concepts
